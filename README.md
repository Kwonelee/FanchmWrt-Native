## 🤔 项目介绍 [![](https://img.shields.io/badge/-项目介绍-FFFFFF.svg)](#-项目介绍-)
**FanchmWrt** 是基于OpenWrt深度开发的 **家用防火墙系统** ，集成 **应用过滤功能** ，功能更强大。

> [!TIP]
> 📢本项目提供了 **一个将第三方 RK 设备适配到 FanchmWrt 官方系统的参考实现** ，支持通过源码编译生成 **`sysupgrade`** 格式固件。该格式支持手动在线刷写升级， **一次适配，持续更新** ，无需重复刷写完整系统
